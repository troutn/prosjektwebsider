Adds a copy button to WordPress code blocks.

== Description ==

This plugin extends the WordPress code block by adding a copy button that allows users to easily copy code snippets to their clipboard.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/copy-code` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
